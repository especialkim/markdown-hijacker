

## 시작하기

- 플러그인 이름 정하기
- Obsidian-sample-plugin 다운받기
- npm install
- `hot-reload` 플러그인 설치하고 활성화하기
- npm run dev

## main.ts

```ts
import { Plugin } from 'obsidian';

export default class ExamplePlugin extends Plugin {
  async onload() {
    console.log('loading plugin')
  }
  async onunload() {
    console.log('unloading plugin')
  }
}
```
